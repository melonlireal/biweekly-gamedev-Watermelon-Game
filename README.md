# Watermelon Game

This is a simple Godot remake of the Watermelon Game (Suika Game) as part of my biweekly minigame challenge. Drop fruits, merge them, and try to reach the watermelon. Built with Godot 4.5.

## Gameplay Features
- Drop fruits into a container by clicking
- Move the mouse to control where the fruit drops
- When two identical fruits touch, they merge into the next fruit type
- Each merge increases your score
- The goal is to create a watermelon, the largest fruit
- The game ends if the fruits drops from the container

How to play: Move the mouse to control position of the fruit, click to drop. Merging two of the same fruit creates the next fruit in the sequence. The game ends if the fruits drop from the container. Try to reach the watermelon for the highest score!
